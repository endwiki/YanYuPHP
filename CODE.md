- 200000 (用户注册成功)
- 210000 (用户登陆成功)

- 200002 (密码必填)

- 200004 (注册用户失败)
- 200005 (用户名或密码错误)
- 200006 (用户不存在)
- 200007 (Token 校验失败)
- 200008 (Token 校验超时)
